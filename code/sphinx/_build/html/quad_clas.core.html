class quad_clas.core.bounds.StatAnalysis(paths, dict_int=None, nn=False, bins=None, truth=False, fit=True, mc_run=None, luminosity=600)[source]¶

Bases: object

This python class object constructs the pdf of the test statistics tc using either the NN reconstructed likelihood ratio or the analytical likelihood ratio. It also computes the expected exclusion limits that follow from a binned analysis, using the asymptotic distributions based on the Asimov dataset.

paths: dict

dictionary that contains the paths to the root directory, output directory and plot directory.

dict_int: dict

dictionary with eft coefficients as values and corresponding lhe-file paths as values.

nn: bool

includes a nn analysis

bins: list

collection of binnings (array_like) for which to run the analysis

truth: bool

includes a truth analysis

fit: bool

should be set to true for each new run/binning. This is needed to find how the expected number of events varies with the Wilson coefficients.

mc_run: int

picks a unique replica

luminosity: float

luminosity of the experiment in pb^{-1}

find_pdf()[source]¶

Computes the mean and standard deviation of the extended likelihood ratio distribution in both the sm and the eft. It then gives the corresponding z-score for each of the eft points in the interpolation dictionary. The results are written to the output directory specified in the paths dictionary.

Depending on whether nn is set to true or not, results are either computed analytically or using the neural network.
